# Personenbezogene Daten
**Beschreibung:** Daten die eine Natürliche Personen bezieht.
- Name
- E-Mail
- Telefonnummer

# Persönlich Daten
**Beschreibung:** Daten die einer person gehören
- Computer Dokumente

# Datensicherheit
**Beschreibung:** Daten sollen nur von Berechtigten Personen gesehen werden.
